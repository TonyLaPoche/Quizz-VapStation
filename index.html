ℹ️ À propos

🎯 Pourquoi ce projet ?

Ce quiz a été créé pour aider les professionnels et passionnés de la vape à maîtriser parfaitement le catalogue Vap'Station.

En tant que vendeur ou amateur éclairé, connaître précisément les arômes de chaque e-liquide est essentiel pour :

  • 🎯 Conseiller efficacement les clients
  • 🔍 Identifier rapidement les produits adaptés
  • 💡 Proposer des alternatives pertinentes
  • 🏆 Développer son expertise produit

🧪 Comment ça marche ?

L'application utilise un système de quiz à choix multiples où vous devez sélectionner tous les arômes corrects pour chaque produit.

Différents modes sont disponibles :

  • Par gamme : Focus sur une gamme spécifique
  • Quiz mélangé : Sélection équilibrée (10 questions)
  • Quiz complet : Tous les produits du catalogue

📱 Application PWA (Progressive Web App)

Cette application peut être installée localement sur tous vos appareils :

  • 💻 PC & Mac : Via Chrome, Edge ou Safari
  • 📱 Android : "Ajouter à l'écran d'accueil"
  • 🍎 iOS : "Ajouter à l'écran d'accueil" dans Safari
  • ⚡ Utilisation hors ligne complète une fois installée

Cherchez le bouton "📱 Installer l'Application" ou l'icône d'installation dans votre navigateur !

📊 Suivi des progrès

Vos performances sont automatiquement sauvegardées localement :

  • 📈 Statistiques globales et par gamme
  • 📋 Historique détaillé des 50 derniers quiz
  • 🎯 Identification des points faibles

🔒 RGPD et confidentialité

Votre vie privée est notre priorité. Cette application est entièrement conforme au RGPD :

  • 🏠 Historique local : scores et stats restent sur votre appareil
  • 🏆 Classement public (optionnel) : si vous partagez un score, code postal + initiales + résultat sont envoyés à Firebase
  • 🔐 Pas de cookies publicitaires
  • 🗑️ Contrôle total : vous pouvez effacer vos données locales à tout moment

Données stockées localement :

  • Résultats de vos quiz (scores, dates, modes)
  • Statistiques de performance par gamme

Données du classement public (si vous choisissez de partager) :

  • Code postal et initiales
  • Score, mode de jeu et date du quiz

💡 Ces données sont stockées dans le localStorage de votre navigateur et ne quittent jamais votre appareil. Elles sont automatiquement supprimées si vous videz le cache de votre navigateur.
